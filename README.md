# 👁️ Project Panoptes: The Surveillance Cult Research Framework

## 📖 Overview
**Project Panoptes** is an open-source sociological research repository and educational framework dedicated to identifying, documenting, and analyzing **Surveillance Cults**. 

While traditional cults rely on physical isolation, modern high-control groups increasingly leverage complex webs of systemic monitoring, digital espionage, coercive control, and algorithmic profiling to dominate their members. This project aims to centralize data on how these groups operate, providing researchers, psychologists, and the public with a comprehensive understanding of panoptic control mechanisms, recruitment strategies, and pathways to extraction.

## ✨ Features
*   **Taxonomy of Surveillance:** A detailed breakdown of the four pillars of high-control monitoring (Technological, Peer-to-Peer, Institutional, and Self-Surveillance).
*   **Global Case Study Database:** Documented analyses of historical and contemporary surveillance cults (e.g., NXIVM, Scientology, Twin Flames Universe).
*   **Algorithmic Recruitment Tracker:** Documentation of how high-control groups use Big Data, targeted advertising, and digital footprinting to groom vulnerable demographics.
*   **Escape & Recovery Architecture:** Research into the unique barriers to escaping digital control (e.g., blackmail via "collateral," financial surveillance) and resources for leaving.

## 🗺️ Roadmap of Future Features
- [ ] **v1.1: Interactive BITE-S Model:** Expanding Steven Hassan's BITE (Behavior, Information, Thought, Emotional) Model to specifically include "Surveillance" as a core tenet.
- [ ] **v2.0: AI Recruitment Analyzer:** An open-source NLP tool that scans text, social media ads, and direct messages for common "love bombing" and cult grooming linguistics.
- [ ] **v3.0: Survivor Data Sanitization Guide:** A technical manual for escaping members on how to scrub spyware, reclaim compromised accounts, and protect against institutional stalking.
- [ ] **v4.0: Legislative Policy Templates:** Drafted proposals for lawmakers to address digital coercion, non-consensual surveillance, and digital blackmail within high-control groups.
