# Methodology & Data Collection

## Selection Criteria
Groups are included if they exhibit **at least two** of the four surveillance pillars and have been documented by reliable sources (court records, investigative journalism, academic research, survivor accounts).

## Sources
We draw exclusively from publicly available information. Each JSON record includes a list of sources with type, publication, and access date. See `data/bibliography.bib` for full academic references.

## Limitations
- Data may be incomplete, especially for active groups where investigations are ongoing.
- The intensity ratings (low/medium/high/extreme) are qualitative assessments based on available evidence; they will be refined as more data becomes available.
- This project does **not** replace professional legal or psychological advice.

## Updates
The dataset is version-controlled. All changes are tracked via pull requests and reviewed by maintainers.
