# Glossary

- **BITE Model**: Behavior, Information, Thought, Emotional control – framework by Steven Hassan.
- **Collateral**: Compromising material (nude photos, confessions) used to enforce obedience.
- **Doxxing**: Public release of private information to intimidate.
- **Love Bombing**: Overwhelming a recruit with affection and attention to create dependency.
- **Panopticon**: A system where subjects believe they are constantly watched.
